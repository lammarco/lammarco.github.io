DeltaPhysics (Prototype)

Role: Solo Developer

A physics-based Metroidvania project in C# Unity2D to deepen my understanding via implementing higher-level features:

  • Editor Tools:
    • Created custom inspector editors ranging from simple button organization to helpers that facilitate map design (e.g. building 2D Sprite Shape curves).
  • Map Design:
    • Determining progression in an interconnected, nonlinear world of physics-related phenomena — while preserving flow as a sequence platforming challenges.
  • Mechanical Depth:
    • Implemented ability upgrades that all interact with the each other, while each having various effects on the player and environments.
  • Mechanical Polish:
    • Worked with (and against) the physics system to improve player experience, such as state-dependent acceleration, adjusted jump curves, input grace times, etc.

Rhythm Game (Discord) Bot

Role: Programmer

I'm finally announcing the beta release of the RG Bot
a discord bot that's a rhythm game database
you can find songs, artists, illustrators, charters, etc
with many contributors & all coding done by @MacroTree this community project has been in the works for over a year
exs: pic.twitter.com/sVoHnB8dXT

— camo (@camoREXXX) January 26, 2022

Written in Python using discord-rewrite. Provides access to rhythm game information via chat commands.

  • Collaborated with community members to compile song chart data into a formatted spreadsheet.
  • Generated database with multi-layer attributes (e.g. subdifficulties) from said spreadsheet.
  • RegEx input processing — including multi-word, non-alphanumeric information (e.g. song titles).
  • Organized data into Discord embeds, sent asynchronously.
